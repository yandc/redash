2021.06.29:
直接部署：
1. 准备python3环境，安装依赖 pip install -r requirements.txt
2. setup-env.sh中配置PostgreSQL和redis地址, 并激活: source setup-env.sh
3. migrate:  ./manage db upgrade
4. 解压静态文件: cd client && tar -zxvf dist.tar.gz
4. 启动服务(server, rq worker, rq scheduler):
   1) server: 参考gunicorn -b 0.0.0.0:5000 --name redash -w 4 redash.wsgi:app
   2) rq worker: supervisor启动，配置在./worker.conf
   3) rq scheduler: supervisor启动./manage.py rq scheduler
   
docker部署：
1. 准备docker环境docker-compose up -d
2. docker-compose.yml中配置PostgreSQL(REDASH_DATABASE_URL)和redis(REDASH_REDIS_URL)
3. 启动服务docker-compose up -d
